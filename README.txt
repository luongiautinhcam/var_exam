Bước 1: Downloads file VAR_EXAM.zip, sau đó giải nén.

Bước 2: Cài đặt các thư viện cần thiết.
- Cài đặt OpenCV:
Để cài đặt OpenCV, bạn có thể chạy lệnh sau trong Command Prompt hoặc Terminal:
pip install opencv-python

- Cài đặt tkinter và PIL:
Tkinter và PIL thường được cài đặt sẵn trong các phiên bản Python, tuy nhiên nếu máy tính của bạn không có sẵn, bạn có thể cài đặt chúng bằng cách chạy các lệnh sau:
pip install tkinter
pip install Pillow

- Cài đặt datetime:
Datetime là một thư viện chuẩn trong Python, nó không yêu cầu cài đặt riêng.

Bước 3: Sau khi cài đặt các thư viện trên chạy file app.py để sử dụng.
